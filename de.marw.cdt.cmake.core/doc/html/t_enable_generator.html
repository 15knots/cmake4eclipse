Enabling CMake buildscript generation

To have your buildscripts generated by cmake,

  1. Open the Project➢ Properties dialog.
  2. Expand the C/C++ Build node and
  3. Select the Tool Chain Editor page.
  4. On the tool chain editor page, select either CMake Builder (portable) or CMake Builder (GNU Make) as the current builder.

See Builders for details on the available builders.
