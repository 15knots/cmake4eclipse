Enabling CMake buildscript generation

To enable the cmake buildscript generation, open the Project > Properties dialog.
Expand the C/C++ Build node and click onto Tool Chain Editor.
On the tool chain editor page, select either CMake Make Builder or CMake Builder (portable) as the current builder.

CMake Make Builder

This builder is nearly identical to the built-in Gnu Make Builder. It always generates makefiles and then invokes the make tool found in the system's executable search path.

CMake Builder (portable)

This builder allows to customize the build tool to be used on the OS-specific property tabs and lets CMake generate the build scripts for the tool of choice. Since CMake does a fairly good job in detecting the absolute file system location of the built tool, the latter can be invoked even if it is not on the executable search path.
