Project Builders

The Cmake4eclipse plugin provides the following Project Builders (selectable in the Tool Chain Editor tab):

CMake Builder

This builder allows you to customize the build output location and makes CMake generate the build scripts.

The builder retrieves the name of the actual build tool from the cache file written by cmake during build-script-generation.

This is the recommended builder.

CMake Builder (deprecated)

This builder is nearly identical to the Gnu Make Builder that comes with CDT. It requires that cmake is configured to generate makefiles and will invoke the make tool found in the system's executable search path ($PATH) to build the project.
It is kept only for backward compatibility with projects set up with cmake4eclipse prior to version 2.0.0 and cannot be selected in the UI any longer.
