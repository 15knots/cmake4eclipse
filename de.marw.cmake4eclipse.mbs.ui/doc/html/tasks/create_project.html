Creating a Project

First of all: Do not try to manually generate an Eclipse project using cmake -G Eclipse CDT4 - Unix Makefiles and import it, as that will make cmake4eclipse unavailable in your project!

As a first start, you may want to try one of the example projects.

As of cmake4eclipse version 4, the plugin comes with a project creation wizard of its own.

Create a Cmake4eclipse Managed Build project.

A Cmake4eclipse Managed Build project is essentially the same as a Managed Build project. Since cmake4eclipse takes the user`s CMakeLists.txt files as the source of truth — in contrast to Managed Build which uses the data entered in lots of project property pages — a Cmake4eclipse Managed Build project does only display the project property pages and tabs that effect cmake`s buildscript generation; in the hope of reducing end user`s confusion.
