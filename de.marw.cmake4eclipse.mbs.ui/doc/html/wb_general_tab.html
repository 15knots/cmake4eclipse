General preferences

These preferences specify commandline options for cmake for each project in the workbench that gets build with cmake4eclipse.

The Default build system corresponds to an implied Build Toolkit that gererates scripts for the specified build-system and finds each of the tools required to build a project on the executable file search path — as specified in the PATH environment variable.
While it is common on Linux to find each tool through the PATH environment variable, this does not hold for users of the windows OS. Therefore, these users are given means to specify the file-system tool-locations of their special needs along with the corresponding build system and overwrite the Default build system.
