Build Toolkit

A Build Toolkit defines

  • the build system to generate,
  • the locations of the tools that actually perform the build and
  • optionally the location of the cmake executable that generates the build scripts.
The locations of the tools is made available through the Eclipse dynamic variable named cmake4eclipse_btk_PATH. The cmake4eclipse_btk_PATH variable might be useful to specify a different PATH environment variable in a debugger configuration.
